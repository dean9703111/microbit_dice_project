## 用Microbit打造魔術心靈骰子

因為在生日的時候收到來自好朋友超鴿，一個充滿工程師風格的驚奇禮物  
Microbit是一個教育性質為主的開發版，上面有各式各樣的感測器，一切都靠你的個人想像力  
因為有過去開發arduino UNO的經驗，所以我這次也是用Arduino做為開發框架  

實作的目標就是利用Microbit本身加速度的感測器＋藍芽模組，結合官方手機App:Blynk  
讓手機上面呈現由Microbit製作而成骰子的數字(可以做為心靈魔術使用XD)  

### 如果你也想要自己做一個心靈骰子，您需要做以下的事前準備:

[這篇文章](https://oranwind.org/-bbc-microbit-qi-shou-shi-arduino-ji-chu-jiao-xue/)詳細的說明arduino開發所需要的一切起手式  
如果你想要對arduino的加速度功能有更深入的探討可以參考[這篇文章](https://sites.google.com/site/wenyumaker2/02micro-bit-yan-jiu/23-micro-bit-with-arduino)  
你可以用一般電子材料行都能找得到的電池盒(3V)作為電源供應(EX:2節7號電池、鈕扣電池盒)，如果你想要穩定及輕量化可以考慮購買[BBC micro:bit -MI: power board 擴充板](http://ipoemaker.com/0110102)

## 以下為Microbit燒入程式後的製作流程

### SETP 1 先在電子材料行購買：
1. 3V並連鈕扣電池盒
2. 2.0 2P的公頭連接線
3. 電火布
4. 鈕扣電池 X 2

### SETP 2 將電池放入鈕扣電池盒：
這裡要特別注意，因為我們買的是並連的電池盒，以我的電池盒為例，這兩顆電池的正極是在下方的  
![image](/img/鈕扣電池盒.jpeg)

### SETP 3 將公頭連接線插入Microbit電源孔：
我購買的規格是2.0 2p公頭連接線，建議購買時可以攜帶自己的arduino板子去店裡確認，因為每塊板子可能規格會有不同
![image](/img/2.0_2p公頭連接線.jpeg)

### SETP 4 確認正負極後看是否正常運行：
這裡要特別注意，一般來說紅色線是正極、黑色線是負極；但是我購買的公頭連接線剛好是相反的(請你仔細看Microbit上面的正負極符號)，所以在跟電池盒連接的時候我是紅黑線各自纏繞  
纏繞完成後打開電池盒電源，你會發現什麼事情都沒有發生，Microbit的板子也沒有亮燈(原本插上usb會亮燈)，請放心這是正常狀況，你要打開手機卻認他是否有正常上線就好了(本範例是使用Blynk的藍芽連線)
![image](/img/正負極確認.jpeg)

### SETP 5 用電火布纏繞：
因為裸露的線材如果沒有風起來很有可能會造成短路，如果短路你的Microbit板子很可能就會報廢，所以請用電火布把這些外露的線材包起來  
![image](/img/用電火布纏繞.jpeg)

### SETP 6 將Microbit放入骰子盒子：
骰子盒子我是用超級土炮的厚紙板製作，裡面塞滿氣泡布做為防震功能，如果追求完美你可以去尋找木製的外盒  
![image](/img/內部.jpeg)
